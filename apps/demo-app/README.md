# Demo App 

Sandbox, prototyping tool, and ultimatey a demo for devs and employers.  

## Run a Build

- install dependencies
   ```zsh
   pnpm i
   ```
- start the metro server
   ```zsh
   npx expo start
   ```
- run a flavor 
  - android
      ```zsh
      npx expo run:ios
      ```
  - ios
      ```js
      npx expo run:android
      ```
